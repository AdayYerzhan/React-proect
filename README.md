
### Это веб приложение аналог твиттера (Twitter).
 
 Оно выполняет следующие функций:
 - Постить новые записи
 - Лайкать записи
 - Выделять избарнные
 - Фильтровать записи
 - Удалять записи
 - Поиск по записям
 - Добавлять записи
